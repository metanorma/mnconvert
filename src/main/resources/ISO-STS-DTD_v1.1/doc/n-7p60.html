International Organization for Standardization (ISO)

vol

Volume of Related Article

Volume of the journal in which a related journal article exists.

Usage

The @vol attribute can be used, along with the @journal-id, @issue, and @page attributes, to provide the metadata for a related journal article.

Attribute Values

In Element

<related-article>
Value Meaning
Data characters Text, numbers, or special characters.
Restriction: This attribute may be specified if the element is used.
