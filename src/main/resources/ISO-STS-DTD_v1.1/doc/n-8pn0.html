International Organization for Standardization (ISO)

rid

Reference to an Identifier

Holds the value of the identifier of an associated element; used for linking related elements (for example, <term> to a <def>, <bio> to a <contrib>).

Attribute Values

In Elements

<milestone-end>
<milestone-start>
Value Meaning
An identifier (IDREF) Points to the existing identifier
Restriction: This attribute may be specified if the element is used.

In Elements

<aff>, <award-id>, <bio>, <contrib>, <def>, <funding-source>, <sig>, <sig-block>, <term>, <xref>
Value Meaning
One or more identifiers (IDREFS) Points to one or more existing identifiers.
Restriction: This attribute may be specified if the element is used.
