ISO Standards Tag Set (ISOSTS) version 1.1: A customization of NISO JATS version 0.4

International Organization for Standardization (ISO)

<standard>

Standard or Standards-Related Document

The root element for the encoding of any standard or standards document.

Remarks

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Attributes

dtd-version Version of the Tag Set (DTD)
specific-use Specific Use
xmlns:mml MathML Namespace Declaration
xmlns:tbx TBX Namespace Declaration
xmlns:xlink XLink Namespace Declaration
xmlns:xsi XML Schema Namespace Declaration
xml:lang Language

Model Description

The following, in order:

  • <front> Front Matter
  • <body> Body of the Document
  • <back> Back Matter, zero or one

This top-level element may not be contained in any other elements.
