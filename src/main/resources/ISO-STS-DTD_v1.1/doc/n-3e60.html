style

Style (XHTML table model; MathML 2.0 Tag Set)

This attribute can be used in several contexts:

  • With the styled content element (<styled-content>) to record HTML classes or similar styling information.
  • Within many of the elements within an XHTML 1.1 Table (<table>), this attribute should be used to record styling. Use W3C documentation for the XHTML1.1 table model (http://www.w3.org/TR/xhtml-modularization/abstract_modules.html#s_tablemodule).
  • Within the <mml:math>, use documentation for the Mathematical Markup Language (MathML) 2.0 Tag Set (http://www.w3.org/TR/MathML2).

Remarks

To name the concrete style or rendering to be applied to the element

When on a Styled Special (Subject) Content element, the value should be as defined in the W3C XHTML 1.1 Tag Set.

Attribute Values

In Elements

<caption>, <col>, <colgroup>, <mml:math>, <styled-content>, <table>, <tbody>, <td>, <tfoot>, <th>, <thead>, <tr>
Value Meaning
Data characters Text, numbers, or special characters.
Restriction: This attribute may be specified if the element is used.
