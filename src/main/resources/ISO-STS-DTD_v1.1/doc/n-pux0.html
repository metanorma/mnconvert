International Organization for Standardization (ISO)

<page-count>

Page Count

This element is used in two instances:

  • the number of pages in a print article, counting each page or partial page as one (Electronic articles do not have page counts.)
  • the page count on which the pricing of the standard is based; appears on the back cover of the standard

Remarks

This element is part of the base JATS journal publishing vocabulary; its definition and documentation have been modified to reflect its use in the ISOSTS customization of JATS.

Attribute

count Count

Model Description

This is an EMPTY element

This element may be contained in:

<iso-meta>, <nat-meta>, <reg-meta>
