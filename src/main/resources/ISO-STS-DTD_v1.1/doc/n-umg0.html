International Organization for Standardization (ISO)

originator

Originator

Flags material added to a standard by another standards body (e.g. in an adoption).

Remarks

When this attribute is not specified on an element, the implication is that the content of the element originated with the standards body identified in the front matter as responsible for the document (unless otherwise specified on a child or descendant element). Material added as part of an adoption should be marked as an addition by using this attribute and indicating who is responsible for the addition.

Attribute Values

In Elements

<address>, <alternatives>, <array>, <boxed-text>, <chem-struct-wrap>, <def-list>, <disp-formula>, <disp-formula-group>, <disp-quote>, <fig>, <fig-group>, <fn-group>, <glossary>, <graphic>, <list>, <nat-meta>, <media>, <non-normative-example>, <non-normative-note>, <p>, <preformat>, <ref-list>, <related-article>, <related-object>, <reg-meta>, <sec>, <speech>, <statement>, <supplementary-material>, <table-wrap>, <table-wrap-group>, <term-sec>, <verse-group>
Value Meaning
Data characters Organization responsible for the addition of the material marked.
Restriction: This attribute may be specified if the element is used.
