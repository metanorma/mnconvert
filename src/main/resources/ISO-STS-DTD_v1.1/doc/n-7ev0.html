ISO Standards Tag Set (ISOSTS) version 1.1: A customization of NISO JATS version 0.4

International Organization for Standardization (ISO)

<release-version>

Release Version

Code indicating the maturity level of the document.

Remarks

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Expected values for ISO standards include:

WD

Working Draft

CD

Committee Draft

DIS

Draft International Standard

FDIS

Final Draft

IS

International Standard (this value is also used for later corrected editions and alternate language versions of an IS)

Other values may be used when the ISOSTS vocabulary is used in other standards-develoment contexts.

Model Description

Text, numbers, or special characters, zero or more

This element may be contained in:

<doc-ident>
