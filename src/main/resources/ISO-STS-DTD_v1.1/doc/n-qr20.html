<author-comment>

Author Comment

Textual material associated with a contributor.

Attributes

content-type Type of Content
id Identifier
specific-use Specific Use
xml:lang Language

Model Description

The following, in order:

  • <title> Title, zero or one
  • One or more of any of:
    • <p> Paragraph
    • <non-normative-note> Non-normative Note
    • <non-normative-example> Non-normative Example

This element may be contained in:

<collab>, <contrib>, <contrib-group>

Example

...
<contrib-group>    
<contrib contrib-type="author">
<name><surname>Young</surname>
<given-names>John G.</given-names></name>
<role>consultant physician</role>
<xref ref-type="aff" rid="RoyalInf"/>

Group</p></author-comment>
</contrib>

of Geriatric Medicine, Royal Infirmary, Glasgow

</contrib-group>
