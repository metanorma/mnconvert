International Organization for Standardization (ISO)

<is-proof>

Is-Proof Flag

A signal that the draft document is being processed as proof. No ballot is required and so a special coverpage must be produced.

Remarks

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Model Description

This is an EMPTY element

This element may be contained in:

<iso-meta>

Example

               
<standard>
  <front>
    <iso-meta>
      <title-wrap xml:lang="en">...</title-wrap>
      <doc-ident>
        <sdo>ISO</sdo>
        <proj-id>?????</proj-id>
        <language>en</language>
        <release-version>DIS</release-version>
        <urn>urn:iso:std:iso:2560:ed-3:v1:en</urn>
      </doc-ident>
      <std-ident>...</std-ident>      
      ...
      <release-date>2011-07-15</release-date>
      <comm-ref>ISO/TC 44/SC 3</comm-ref>
      <secretariat>ANSI</secretariat>
      <ics>25.160.20</ics>
      <page-count count="29"/>
      <is-proof/>
    </iso-meta>
    
  </front>
  <body>...</body>
</standard>
