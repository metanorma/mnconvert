International Organization for Standardization (ISO)

<aff-alternatives>

Affiliation Alternatives

Container element to hold two or more representations of a single affiliation (for example, the name of a university in two or more languages such as both German and English).

Attribute

id Identifier

Model Description

<aff> Affiliation, one or more

This element may be contained in:

<collab>, <contrib>, <contrib-group>, <person-group>

Example

...
<supplement>
<contrib-group>
<contrib rid="aff1">
<name><surname>Smythe-Barring</surname>
<given-names>John</given-names>
</name>
<aff-alternatives>
<aff id="aff1">
<institution xml:lang="fr">Mus&eacute;e national de la linguistique</institution>
</aff>
<aff id="aff-en">
<institution xml:lang="en">National Museum of Linguistics</institution>
</aff>
</aff-alternatives>
</contrib>
</contrib-group>
...
</supplement>
...
