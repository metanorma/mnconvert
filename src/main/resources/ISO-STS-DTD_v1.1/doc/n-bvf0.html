count

Count

Number of specific objects that appear in the document; which objects included in the count is indicated by the element name (for example, <page-count>).

Attribute Values

In Element

<page-count>
Value Meaning
A word A word containing only letters, numbers, and a few special characters.
Restriction: This attribute must be specified if the element is used.
