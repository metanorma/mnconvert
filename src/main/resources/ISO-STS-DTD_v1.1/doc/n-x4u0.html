<media>

Media Object

Description of and pointer to an external file that holds a media object (for example, an animation, a movie).

Attributes

content-type Type of Content
id Identifier
mime-subtype Mime Subtype
mimetype Mime Type
orientation Orientation
originator Originator
position Position
specific-use Specific Use
xlink:actuate Actuating the Link
xlink:href Href (Linking Mechanism)
xlink:role Role of the Link
xlink:show Showing the Link
xlink:title Title of the Link
xlink:type Type of Link
xmlns:xlink XLink Namespace Declaration
xml:lang Language

Model Description

Any combination of:

  • <alt-text> Title, Alternate
  • <long-desc> Long Description
  • <email> Email Address
  • <ext-link> External Link
  • <uri> Uniform Resource Indicator (URI)
  • <object-id> Object Identifier
  • <label> Label of a Figure, Reference, Etc.
  • <caption> Caption of a Figure, Table, etc.
  • <attrib> Attribution
  • <permissions> Permissions

This element may be contained in:

<ack>, <alternatives>, <app>, <app-group>, <array>, <bio>, <body>, <boxed-text>, <chem-struct>, <chem-struct-wrap>, <disp-formula>, <disp-quote>, <fig>, <fig-group>, <glossary>, <license-p>, <named-content>, <non-normative-example>, <non-normative-note>, <notes>, <p>, <ref-list>, <sec>, <sig>, <sig-block>, <styled-content>, <supplementary-material>, <table-wrap>, <td>, <term>, <term-display>, <term-sec>, <th>
