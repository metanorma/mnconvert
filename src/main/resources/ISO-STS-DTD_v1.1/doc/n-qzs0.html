International Organization for Standardization (ISO)

<suppl-type>

Supplement Type

Identifies what kind of alteration of addition is represented by this document (for documents which are supplements to a standard).

Remarks

Usage. This element is applicable to supplements only; for other documents it may be omitted; if present it should be empty. Expected values are:

amd

Amendment

cor

Technical Corrigendum

add

Addendum (will probably not be encountered in new documents; some published addenda are still valid, but addenda are no longer being published)

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Model Description

Text, numbers, or special characters, zero or more

This element may be contained in:

<std-ident>, <std-ref>

Example

               
...
<std-ident>
<originator>CEN</originator>
<doc-type>EN</doc-type>
<doc-number>3951</doc-number>
<part-number>2</part-number>
<edition>1</edition>
<version>1</version>
<suppl-type>1</suppl-type>
<suppl-number>1</suppl-number>
<suppl-version>2</suppl-version>
</std-ident>
...
