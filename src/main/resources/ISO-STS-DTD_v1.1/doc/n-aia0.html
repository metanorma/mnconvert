ISO Standards Tag Set (ISOSTS) version 1.1: A customization of NISO JATS version 0.4

International Organization for Standardization (ISO)

<label>

Label of a Figure, Reference, Etc.

Number and/or prefix word placed at the beginning of display elements (for example, equation, statement, figure).

Attributes

alt Alternative Text
xml:lang Language

Model Description

Any combination of:

  • Text, numbers, or special characters
  • <bold> Bold
  • <italic> Italic
  • <monospace> Monospace Text (Typewriter Text)
  • <num> Number
  • <overline> Overline
  • <roman> Roman
  • <sans-serif> Sans Serif
  • <sc> Small Caps
  • <strike> Strike Through
  • <underline> Underline
  • <alternatives> Alternatives for Processing
  • <inline-graphic> Inline Graphic
  • <private-char> Private Character (Custom or Unicode™)
  • <chem-struct> Chemical Structure (Display)
  • <inline-formula> Formula, Inline
  • <named-content> Named Special (Subject) Content
  • <styled-content> Styled Special (Subject) Content
  • <fn> Footnote
  • <target> Target of an Internal Link
  • <tbx:entailedTerm> Entailed Term (TBX Tag Set)
  • <xref> X(cross) Reference
  • <sub> Subscript
  • <sup> Superscript

This element may be contained in:

<ack>, <aff>, <app>, <app-group>, <array>, <back>, <bio>, <boxed-text>, <chem-struct>, <chem-struct-wrap>, <def-list>, <disp-formula>, <disp-formula-group>, <disp-quote>, <element-citation>, <fig>, <fig-group>, <fn>, <fn-group>, <glossary>, <graphic>, <list>, <list-item>, <media>, <mixed-citation>, <non-normative-example>, <non-normative-note>, <notes>, <ref>, <ref-list>, <sec>, <statement>, <sub-part>, <supplementary-material>, <table-wrap>, <table-wrap-group>, <term-sec>, <verse-group>
