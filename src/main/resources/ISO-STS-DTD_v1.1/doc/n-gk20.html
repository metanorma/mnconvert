International Organization for Standardization (ISO)

<annex-type>

Type of Annex

Denotes whether an annex is normative or informative

Remarks

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Expected values are “(normative)” and “(informative)”; other values are valid and may be needed in contexts other than ISO work.

Model Description

Text, numbers, or special characters

This element may be contained in:

<app>
