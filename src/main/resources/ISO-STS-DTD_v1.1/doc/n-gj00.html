ISO Standards Tag Set (ISOSTS) version 1.1: A customization of NISO JATS version 0.4

International Organization for Standardization (ISO)

<aff>

Affiliation

Name of a institution or organization such as a university or corporation.

Attributes

content-type Type of Content
id Identifier
rid Reference to an Identifier
specific-use Specific Use
xml:lang Language

Model Description

Any combination of:

  • Text, numbers, or special characters
  • <addr-line> Address Line
  • <country> Country: in an Address
  • <fax> Fax Number: in an Address
  • <institution> Institution Name: in an Address
  • <phone> Phone Number: in an Address
  • <email> Email Address
  • <ext-link> External Link
  • <uri> Uniform Resource Indicator (URI)
  • <inline-supplementary-material> Inline Supplementary Material
  • <related-article> Related Article Information
  • <related-object> Related Object Information
  • <break> Line Break
  • <bold> Bold
  • <italic> Italic
  • <monospace> Monospace Text (Typewriter Text)
  • <num> Number
  • <overline> Overline
  • <roman> Roman
  • <sans-serif> Sans Serif
  • <sc> Small Caps
  • <strike> Strike Through
  • <underline> Underline
  • <label> Label of a Figure, Reference, Etc.
  • <fn> Footnote
  • <target> Target of an Internal Link
  • <tbx:entailedTerm> Entailed Term (TBX Tag Set)
  • <xref> X(cross) Reference
  • <sub> Subscript
  • <sup> Superscript

This element may be contained in:

<aff-alternatives>, <collab>, <contrib>, <contrib-group>, <person-group>
