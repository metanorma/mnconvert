content-type

Type of Content

Identifies a subject or type of content that makes this element semantically distinct.

Remarks

When this attribute is used on <app> elements, the expected values are:

norm-annex

Normative annex / Annexe normative

inform-annex

Informative annex / Annex informative

bibl

Bibliography, Bibliographie

Attribute Values

In Element

<named-content>
Value Meaning
Data characters Text, numbers, or special characters.
Restriction: This attribute must be specified if the element is used.
