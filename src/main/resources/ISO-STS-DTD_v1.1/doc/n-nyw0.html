<object-id>

Object Identifier

Unique identifier (such as a DOI or URI) for a component within an article (for example, for a figure or a table).

Attributes

content-type Type of Content
pub-id-type Type of Publication Identifier
specific-use Specific Use

Model Description

Text, numbers, or special characters

This element may be contained in:

<boxed-text>, <chem-struct-wrap>, <element-citation>, <fig>, <graphic>, <media>, <mixed-citation>, <related-article>, <related-object>, <supplementary-material>, <table-wrap>
