International Organization for Standardization (ISO)

<doc-type>

Document Type

Indicates what type of document this is.

Remarks

Usage. The expected values for this element are:

guide

document giving rules, orientation, advice or recommendations relating to international standardization

is

international standard (default value)

isp

international standard profile

iwa

international workshop agreement

pas

publicly available specification; document published to respond to an urgent market need, representing either a) a consensus in an organization external to ISO or IEC, or b) a consensus of the experts within a working group

r

recommendation (this value is now obsolete and will not be encountered with new documents)

tr

technical report; document published by a standards development organization containing collected data of a different kind from that normally published as an International Standard or Technical Specification

ts

technical specification; document for which there is the future possibility of agreement on an International Standard, but for which at present either the required support for approval as an International Standard cannot be obtained, or there is doubt on whether consensus has been achieved, or the subject matter is still under technical development, or there is another reason precluding immediate publication as an International Standard

tta

technology trends assessment
These values should be used when appropriate, but other values may and should be used when needed; the element is not restricted to this set of values.

Conversion Note: It may be necessary to infer the value of this element from the label at the beginning of the title page of the document.

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Related Elements

Model Description

Text, numbers, or special characters, zero or more

<std-ident>, <std-ref>
