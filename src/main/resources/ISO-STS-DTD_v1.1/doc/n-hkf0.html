International Organization for Standardization (ISO)

collab-type

Type of Collaboration

Type or role of the collaborators; what function they played in the publication (for example, “ translators ”, “ editors ”).

Attribute Values

In Element

<collab>
Value Meaning
Data characters Text, numbers, or special characters.
Restriction: This attribute may be specified if the element is used.

Suggested usage

Although the Tag Set does not enforce any values here, to preserve what the publisher has established, suggested values for this attribute include:

assignee

Group or company to whom a patent is awarded

authors

Content creators

editors

Content editors

compilers

Person or group that has put together a composite work from multiple sources

guest-editors

A group of editors that has been invited to edit all or part of a work

inventors

Idea, software, or machine creators

translators

Person or group that has translated the cited work from one language into another
