International Organization for Standardization (ISO)

<caption>

Caption of a Figure, Table, etc.

Container element for the textual description that is associated with an object (for example, with a <fig> or <table>).

Attributes

content-type Type of Content
id Identifier
specific-use Specific Use
style Style (XHTML table model; MathML 2.0 Tag Set)
xml:lang Language

Model Description

The following, in order:

  • <title> Title, zero or one
  • Any combination of:
    • <p> Paragraph
    • <non-normative-note> Non-normative Note
    • <non-normative-example> Non-normative Example

This element may be contained in:

<boxed-text>, <chem-struct-wrap>, <disp-formula-group>, <fig>, <fig-group>, <graphic>, <media>, <supplementary-material>, <table-wrap>, <table-wrap-group>
