supplement-type

Type of Supplement

Indicates the type of supplement or the reason for the supplement (for example, “ conference ” for a supplement containing information from a conference symposium).

Remarks

Indicates the type of supplement or reason for the supplement (if the publisher has recorded that information), for example:

issue

for supplements to a particular journal issue

conference

for supplements containing information from a conference, symposium, or other gathering

periodic

for supplements issued at regular intervals

topic

for supplements concerning a particular subject area

Attribute Values

In Element

<supplement>
Value Meaning
Data characters Text, numbers, or special characters.
Restriction: This attribute may be specified if the element is used.
