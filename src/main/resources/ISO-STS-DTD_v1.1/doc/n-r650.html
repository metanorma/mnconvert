International Organization for Standardization (ISO)

<edition>

Edition, Cited

Edition number of a standard; may also used for edition number of a cited publication.

Remarks

This element is used in the metadata for the standard reference <std-ident> and may also be used in bibliographic references.

Attributes

content-type Type of Content
specific-use Specific Use
xml:lang Language

Model Description

Text, numbers, or special characters, zero or more

This element may be contained in:

<element-citation>, <mixed-citation>, <related-article>, <related-object>, <std-ident>, <std-ref>

Example

              
...
    <iso-meta>
      ...
      <std-ident>
        <originator>ISO</originator>
        <doc-type>is</doc-type>
        <doc-number>3951</doc-number>
        <part-number>2</part-number>
        <edition>1</edition>
        <version>1</version>
        ...
      </std-ident>
      <content-language>en</content-language>
      <std-ref type="dated">ISO 3951-2:2006</std-ref>
      <std-ref type="undated">ISO 3951-2</std-ref>
      ...
    </iso-meta>
  ...
