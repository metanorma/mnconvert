ISO Standards Tag Set (ISOSTS) version 1.1: A customization of NISO JATS version 0.4

International Organization for Standardization (ISO)

<body>

Body of the Document

Main textual portion of the document that conveys the main part of the normative content.

Remarks

The subdivisions of the standard's body are tagged as <sec> elements; ISOSTS does not provide specialized elements for standardized section types like Scope, References, Terms and Definitions, etc. (Sections of these types can be recognized by their titles and by the value of their @sec-type attribute.)

Attribute

specific-use Specific Use

Related Elements

A standard or standards-related document <standard> may be divided into several components:

  1. the <front> (the metadata or header information, which contains metadata specific to one or more of the responsible standards-development organizations tagged as <iso-meta>, <reg-meta>, or <nat-meta>);
  2. the <body> (the textual and graphical content of the standard);
  3. any <back> (any ancillary information such as a glossary, reference list, or appendix);

Model Description

The following, in order:

  • Any combination of:
    • <address> Address/Contact Information
    • <alternatives> Alternatives for Processing
    • <array> Array (Simple Tabular Array)
    • <boxed-text> Boxed Text
    • <chem-struct-wrap> Chemical Structure Wrapper
    • <fig> Figure
    • <fig-group> Figure Group
    • <graphic> Graphic
    • <media> Media Object
    • <non-normative-note> Non-normative Note
    • <non-normative-example> Non-normative Example
    • <preformat> Preformatted Text
    • <supplementary-material> Supplementary Material
    • <table-wrap> Table Wrapper
    • <table-wrap-group> Table Wrapper Group
    • <disp-formula> Formula, Display
    • <disp-formula-group> Formula, Display Group
    • <p> Paragraph
    • <def-list> Definition List
    • <list> List
    • <mml:math> Math (MathML 2.0 Tag Set)
    • <related-article> Related Article Information
    • <related-object> Related Object Information
    • <disp-quote> Quote, Displayed
    • <speech> Speech
    • <statement> Statement, Formal
    • <verse-group> Verse Form for Poetry
  • Any combination of:
    • <sec> Section
    • <term-sec> Terminology Section
    • <sub-part> Sub-part of Standard or Standards-Related Document
  • <sig-block> Signature Block, zero or one

This element may be contained in:

<standard>, <sub-part>
