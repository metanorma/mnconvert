International Organization for Standardization (ISO)

<contrib>

Contributor

Container element for information about a single author, editor, or other contributor.

Attributes

contrib-type Type of Contribution
corresp Corresponding Author
deceased Deceased
equal-contrib Equal Contribution
id Identifier
rid Reference to an Identifier
specific-use Specific Use
xlink:actuate Actuating the Link
xlink:href Href (Linking Mechanism)
xlink:role Role of the Link
xlink:show Showing the Link
xlink:title Title of the Link
xlink:type Type of Link
xmlns:xlink XLink Namespace Declaration

Model Description

The following, in order:

  • Any combination of:
    • <anonymous> Anonymous
    • <collab> Collaborative (Group) Author
    • <name> Name of Person (Structured)
    • <name-alternatives> Name Alternatives
  • <degrees> Degree(s), zero or more
  • Any combination of:
    • <address> Address/Contact Information
    • <aff> Affiliation
    • <aff-alternatives> Affiliation Alternatives
    • <author-comment> Author Comment
    • <bio> Biography
    • <email> Email Address
    • <ext-link> External Link
    • <on-behalf-of> On Behalf of
    • <role> Role or Function Title of Contributor
    • <uri> Uniform Resource Indicator (URI)
    • <xref> X(cross) Reference

This element may be contained in:

<contrib-group>

Example

...
<supplement>
<contrib-group>
<contrib rid="aff1">
<name><surname>Smythe-Barring</surname>
<given-names>John</given-names>
</name>
<aff-alternatives>
<aff id="aff1">
<institution xml:lang="fr">Mus&eacute;e national de la linguistique</institution>
</aff>
<aff id="aff-en">
<institution xml:lang="en">National Museum of Linguistics</institution>
</aff>
</aff-alternatives>
</contrib>
</contrib-group>
...
</supplement>
...
