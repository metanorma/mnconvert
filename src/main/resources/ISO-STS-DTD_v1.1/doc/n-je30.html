International Organization for Standardization (ISO)

<contrib-group>

Contributor Group

Container element for one or more contributors and information about those contributors.

Attributes

content-type Type of Content
id Identifier
specific-use Specific Use

Model Description

The following, in order:

  • <contrib> Contributor, one or more
  • Any combination of:
    • <address> Address/Contact Information
    • <aff> Affiliation
    • <aff-alternatives> Affiliation Alternatives
    • <author-comment> Author Comment
    • <bio> Biography
    • <email> Email Address
    • <ext-link> External Link
    • <on-behalf-of> On Behalf of
    • <role> Role or Function Title of Contributor
    • <uri> Uniform Resource Indicator (URI)
    • <xref> X(cross) Reference

This element may be contained in:

<collab>, <supplement>

Example

...
<supplement>
<contrib-group>
<contrib rid="aff1">
<name><surname>Smythe-Barring</surname>
<given-names>John</given-names>
</name>
<aff-alternatives>
<aff id="aff1">
<institution xml:lang="fr">Mus&eacute;e national de la linguistique</institution>
</aff>
<aff id="aff-en">
<institution xml:lang="en">National Museum of Linguistics</institution>
</aff>
</aff-alternatives>
</contrib>
</contrib-group>
...
</supplement>
...
