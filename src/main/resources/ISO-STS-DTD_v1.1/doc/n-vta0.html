ISO Standards Tag Set (ISOSTS) version 1.1: A customization of NISO JATS version 0.4

International Organization for Standardization (ISO)

<iso-meta>

ISO Metadata

Container for ISO-specific metadata describing the standard; contains all the information needed to produce the title page and back cover of the document.

Remarks

This element contains one or more document titles (typically English and French), a document-identity section, the standard reference for the document, ICS classification information, an indication of the number of pages in the document (for pricing purposes), a flag indicating the need for special proof-processing, and optionally additional custom metadata.

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Attribute

id Identifier

Related Elements

Metadata required by other bodies working with a document may be recorded in the <reg-meta> or the <nat-meta> element.

Model Description

The following, in order:

  • <title-wrap> Title Wrapper, one or more
  • <doc-ident> Document Identification Section
  • <std-ident> Standard Reference
  • <content-language> Document Language, one or more
  • <std-ref> Standard Reference, one or more
  • <doc-ref> Document Reference
  • <pub-date> Publication Date, zero or one
  • <release-date> Release Date
  • <meta-date> Other Metadata Date, zero or more
  • <comm-ref> Committee, Subcommittee, Working Group
  • <secretariat> Secretariat, one or more
  • <ics> International Classification for Standards, zero or more
  • <page-count> Page Count
  • <is-proof> Is-Proof Flag, zero or one
  • <std-xref> Cross Reference to Other Standard, zero or more
  • <permissions> Permissions, zero or more
  • <custom-meta-group> Custom Metadata Group, zero or more

This element may be contained in:

<front>

        <compl>Classification</compl>

	  &Eacute;lectrodes enrob&eacute;es pour le soudage manuel
	  &agrave; l'arc des aciers non alli&eacute; et des aciers
	  &agrave; grains fins &mdash; Classification</full>
      </title-wrap>
      <doc-ident>...</doc-ident>

      ...
    </iso-meta>
  </front>
  <body>...</body>
