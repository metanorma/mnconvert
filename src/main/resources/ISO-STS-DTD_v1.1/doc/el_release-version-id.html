Release version id

Technical key present in the CEN PROJEXDATA which identifies a standard release version. Could be useful for linking to other meta-data sources i.e. PROJEXDATA.

Remarks

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Attribute

id Identifier

Model Description

Text, numbers, or special characters, zero or more
