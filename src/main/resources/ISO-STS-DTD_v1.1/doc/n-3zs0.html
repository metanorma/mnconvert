International Organization for Standardization (ISO)

<suppl-number>

Supplement Number

Gives the number of the supplement (for documents which are supplements to a standard).

Remarks

Usage. The value of the element is a sequence of digits. For documents which are not supplements, the element may be omitted, or may be present and empty. For supplements, the default value is “1”, so a supplement with an empty <suppl-number> element will be treated as having the number 1.

This element is part of the ISOSTS customization of the JATS journal publishing vocabulary; it is not part of the base vocabulary.

Model Description

Text, numbers, or special characters, zero or more

This element may be contained in:

<std-ident>, <std-ref>

Example

               
...
<std-ident>
<originator>CEN</originator>
<doc-type>EN</doc-type>
<doc-number>3951</doc-number>
<part-number>2</part-number>
<edition>1</edition>
<version>1</version>
<suppl-type>1</suppl-type>
<suppl-number>1</suppl-number>
<suppl-version>2</suppl-version>
</std-ident>
...
