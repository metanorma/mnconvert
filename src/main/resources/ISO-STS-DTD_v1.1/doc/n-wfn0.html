International Organization for Standardization (ISO)

pub-type

Type of Publication

Indicates the medium or publication stage of the document (for example, “ electronic-preprint ”).

Attribute Values

In Elements

<issn>
<pub-date>
Value Meaning
Data characters Text, numbers, or special characters.
Restriction: This attribute may be specified if the element is used.

Suggested usage

Although designed to accept any text as its value, potential values for this attribute might include:

epub

Electronic publication (usually web, but also includes CD-ROM or other electronic only distribution)

ppub

Print publication

epub-ppub

Both print and electronic publications

epreprint

Electronic preprint dissemination

ppreprint

Print preprint dissemination

ecorrected

Corrected in electronic

pcorrected

Corrected in print

eretracted

Retracted in electronic

pretracted

Retracted in print
