elocation-id

Elocation Identifier

Bibliographic identifier for a document that does not have traditional printed page numbers.

Remarks

Performs the same function that the element Electronic Location Identifier does for a regular article. That is, provides an identifier such as a DOI for the related article.

Attribute Values

In Element

<related-article>
Value Meaning
Data characters Text, numbers, or special characters.
Restriction: This attribute may be specified if the element is used.
